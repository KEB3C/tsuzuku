# 🩺 Tsuzuku

Sistema web desenvolvido em Flask para auxiliar na organização de informações clínicas e geração de resumos médicos.

## ✨ Funcionalidades

- Cadastro de usuários
- Login seguro
- Dashboard do usuário
- Criação de resumos clínicos
- Histórico de resumos
- Upload de documentos
- Interface responsiva

## 🛠 Tecnologias utilizadas

- Python
- Flask
- SQLite
- HTML5
- CSS3
- JavaScript

## 📂 Estrutura do projeto

```
app.py
database.py
templates/
static/
uploads/
```

## 🚀 Como executar

1. Clone este repositório.
2. Instale as dependências.
3. Execute:

```bash
python app.py
```

4. Acesse:

```
http://localhost:5000
```
